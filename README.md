# Money Buddy - Personal Finance Tracker PWA

A personal finance tracking app that works as a Progressive Web App.

## Setup on GitHub Pages

1. Create a new repository on GitHub
2. Upload ALL files from this folder
3. Go to Settings → Pages
4. Set Branch to "main" and folder to "/ (root)"
5. Save and wait 1-2 minutes
6. Visit: `https://YOUR-USERNAME.github.io/REPO-NAME/`

## Add to Home Screen (iOS)

1. Open your GitHub Pages URL in Safari
2. Tap Share → Add to Home Screen
3. Name it and tap Add

## Files

- `index.html` - Main page
- `app.js` - The bundled app code
- `manifest.json` - PWA configuration
- `sw.js` - Service worker for offline support
- `icon-192.png` - App icon (192x192)
- `icon-512.png` - App icon (512x512)

## Data Storage

Your data is saved in localStorage and persists between sessions.
Use the Backup feature to export your data!
